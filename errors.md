
# List of error codes which can be sent in the YAML header

| code | description            |
|------|------------------------|
| 0    | ok                     |
| 1    | file_not_exists        |
| 2    | unknown                |
| 3    | invalid_command        |
| 4    | invalid_file_type      |
| 5    | access denied          |
| 6    | invalid content length |
| 7    | file_exists            |
