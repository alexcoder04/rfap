
# List of commands

Note: every command is 4 bytes long.

| command | hex value  | description                                           |
|---------|------------|-------------------------------------------------------|
| ping    | 0x00000000 | sign that connection is still alive                   |
| read    | ...        | get file / directory content                          |
| info    | ...        | get file/directory information (size, file type, ...) |
| ...     | ...        | ...                                                   |
